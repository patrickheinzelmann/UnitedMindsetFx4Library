public static function validateClassStructure(objectToValidate:Object, primaryClass:Class, secondaryClass:Class = null):Boolean

Validates that an object is of a specific class type. If the objectToValidate is an Array or ArrayCollection, the function will use the secondaryClass (if it exists) to check the objects in the array. This function only goes one level deep, not appropriate to check tree structures.
